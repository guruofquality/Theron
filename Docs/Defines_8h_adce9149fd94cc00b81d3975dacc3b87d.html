#define THERON_MAX_ACTORS   8192

Limits the maximum number of actors that can be created at once within Theron.

If you want to create a very large number of simultaneous actors then you'll need to change the value of this define. Note that doing so may lead to higher fixed memory overheads, so should be avoided unless really necessary.

Defaults to 8192.

The value of THERON_MAX_ACTORS can be overridden by defining it globally in the build (in the makefile using -D, or in the project preprocessor settings in Visual Studio).
