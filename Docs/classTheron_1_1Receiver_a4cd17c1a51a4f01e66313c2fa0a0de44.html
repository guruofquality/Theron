void Theron::Receiver::Reset ( )
inline

Resets to zero the count of messages received but not yet consumed.

The internal count of the number of messages that have arrived but not yet been consumed (by calls to Wait or Consume) is reset to zero, effectively forgetting about all messages that have previously arrived.
