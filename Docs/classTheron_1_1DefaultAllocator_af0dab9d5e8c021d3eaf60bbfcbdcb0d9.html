void Theron::DefaultAllocator::Free ( void *const  memory)
inlinevirtual

Frees a previously allocated block of contiguous memory.

Free is called by Theron to deallocate its internal allocations, unless the DefaultAllocator is replaced by a custom allocator via AllocatorManager::SetAllocator. The DefaultAllocator may also be used for allocations within user application code, if desired, in which case it can be accessed via AllocatorManager::GetAllocator.

Parameters
memoryPointer to the memory to be deallocated.
Note
The pointer must not be null, and must address an allocated block of memory.
